## Quá trình thực hiện

Bước 1: Thu thập dữ liệu trên website TopCV.

Bước 3. Lọc, làm sạch dữ liệu trên Hadoop bằng Spark. Sau đó lưu thành 2 bản: 1 bản lưu trả về Hadoop, 1 bản gửi lưu vào Elasticsearch.
